= Graphics Module
:toc:
:authors: Cloudberry Pi Foundations

A simple Python library that makes data visualization easier using Matplotlib with a clean API and built-in 2D + 3D graphs.

== Example Usage

[source,python]
----
from graphs import Graphs

g = Graphs(
    title="Animal Balance",
    theme="ggplot"
)

g.pie_chart(
    labels=["Humans", "Animals", "Insects"],
    values=[80, 50, 40]
)

g.bar_graph(
    labels=["Humans", "Animals", "Insects"],
    values=[80, 50, 40]
)

g.horizontal_bar_graph(
    labels=["Humans", "Animals", "Insects"],
    values=[80, 50, 40]
)
----

== Creating a Graph Object

[source,python]
----
from graphs import Graphs

g = Graphs(
    title="My Graph",
    figsize=(8, 5),
    theme="default",
    grid=True
)
----

=== Parameters

* title → Graph title
* figsize → Figure size
* theme → Matplotlib style (e.g. ggplot, default)
* grid → Enable grid (True/False)

== 3D GRAPHS (VALUE-ONLY API)

All 3D graphs use ONLY values (no x/y/z required). The library generates structure automatically.

---

=== 3D Bar Graph

[source,python]
----

---

=== 3D Line Graph (True 3D)

[source,python]

== Full Example

[source,python]
----
from graphs import Graphs

g = Graphs(
    title="Sales Data",
    theme="ggplot"
)

g.bar_graph(
    labels=["India","USA","China"],
    values=[100,200,300]
)

g.bar_3d_graph(
    labels=["India","USA","China"],
    values=[100,200,300]
)

g.mountain_3d_graph(
    values=[5,10,3,12,7]
)

g.save("output.png")
----
